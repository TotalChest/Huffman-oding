# HuffmanCoding

#### Compression of text files by the Huffman algorithm

### Encode:
```bash
$ g++ encode.cpp -o encode
$ ./encode <input_file>  # 'code' and 'table' will appear in the current directory 
```
### Decode:
```bash
$ g++ decode.cpp -o decode
$ ./decode <code> <table> <output_file>
```
